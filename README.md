# 🌾 Rice Leaf Disease Classification Using Deep Learning and Transfer Learning

## 📌 Project Overview

This project focuses on identifying different rice leaf diseases using advanced Deep Learning and Transfer Learning approaches.

The dataset contains rice leaf images belonging to three major disease categories:

- Bacterial Leaf Blight
- Brown Spot
- Leaf Smut

The primary objective of this project is to:

- Analyze the rice leaf image dataset
- Perform exploratory data analysis (EDA)
- Visualize disease categories and image characteristics
- Preprocess image data and perform normalization
- Apply image augmentation techniques
- Train multiple deep learning models
- Perform transfer learning using pre-trained architectures
- Evaluate model performance using advanced classification metrics
- Identify the best-performing model for accurate rice leaf disease prediction

---

## 🎯 Problem Statement

Develop an intelligent image classification system capable of accurately identifying rice leaf diseases from image data.
